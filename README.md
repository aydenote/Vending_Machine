# 주제 : 벤딩 머신

## 동작 설명 :

> 1.  금액 입력 후 입금 클릭  
>     ➡ 입금 전 클릭 이미지 클릭 불가.  
>     ➡ 입금 후 잔액에 금액 표시.
> 2.  메인 화면 콜라 이미지 클릭  
>     ➡ 클릭한 콜라는 하단 get list에 등록.
> 3.  콜라 재고 5개 get list에 등록  
>     ➡ 해당 콜라 매진.  
>     ➡ 매진된 콜라는 클릭 불가.
> 4.  획득 클릭.  
>     ➡ get list에 등록된 콜라 획득.  
>     ➡ 총금액이 소지금보다 높을 경우 획득 불가.
> 5.  획득한 음료 표시  
>     ➡ 성공적으로 획득시 획득한 음료 list에 표시.
> 6.  반환 클릭.  
>     ➡ 잔액은 0원으로 표시되며, 반환금에 금액 추가.
